⚙️ One-time Setup

This app uses Firebase Realtime Database (free) so all your users sync live without needing any platform. Do this once — paste your Firebase config below.

1Go to console.firebase.google.com → Create a project (e.g. "StudyPlanner")
2Click Realtime Database → Create database → Start in test mode
3Go to Project Settings (⚙️) → Your apps → Add web app → Register → Copy the firebaseConfig object
4Paste the entire firebaseConfig = { ... } JSON below and click Save
🔑 Firebase Config

📱 How to distribute to users

1. After setup, the creator saves this HTML file with the config pre-filled
2. Share the HTML file via WhatsApp / Google Drive / any link
3. Users open it in Chrome → tap the menu → "Add to Home Screen"
4. It installs as an app — works offline too, syncs when back online

💡 Tip: After you configure Firebase, use your browser's "Save as" to save this page — the config gets embedded in the saved HTML for easy sharing.
